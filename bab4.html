Dasar Teori

TIPE DATA STRING

Tipe data string adalah tipe data dalam pemrograman yang digunakan untuk menyimpan kumpulan karakter, angka, tanggal, dan waktu. String dapat menampung banyak karakter sekaligus, biasanya hingga 255 karakter. Penulisan data String di Python dan Javascript, diapit oleh tanda petik.


#Contoh penulisan String dengan petik satu
print('Programming Language')

#Contoh penulisan String dengan petik Dua
print("Python Dynamic Programming Language")

#Contoh penulisan String menampilkan petik di dalamnya
print("Andy sedang membawakan acara 'Dies Natalis UWP'")
print('"Apa yang kamu lakukan?" tanya Ibu.')
print("Pak Budi sedang melaksanakan sholat Jum'at")

Output dari contoh penulisan String di atas adalah sebagai berikut:


Programming Language
Python Dynamic Programming Language
Andy sedang membawakan acara 'Dies Natalis UWP'
"Apa yang kamu lakukan?" tanya Ibu.
Pak Budi sedang melaksanakan sholat Jum'at

Mengakses nilai dalam String

Untuk mengakses substring, caranya sama dengan mengakses tipe data List gunakan tanda kurung siku dan panggil indeks elemennya. Sebagai contoh :.


data = 'Python Dynamic Programming Language'
print(data[3:])
print(data[:9])
print(data[-12:-2])

Output dari contoh akses substring di atas adalah sebagai berikut:


hon Dynamic Programming Language
Python Dy
ing Langua

Karakter Escape Python

Daftar karakter escape atau karakter non-printable yang dapat diwakili/ditulis dengan awalan notasi backslash, berikut beberapa contohnya.


print("Pemrograman Modular \n Pemrograman Dinamis")  #contoh penggunaan Enter (new line)
print("\t Object oriented programming") #contoh penggunaan tab

Python memiliki metode built-in untuk memanipulasi string, berikut ini adalah beberapa contohnya.

Metode Penjelasan
capitalize() Meng-kapitalkan huruf pertama string
isalnum() Mengembalikan true jika string memiliki minimal 1 karakter dan semua karakternya alfanumerik dan false sebaliknya.
isalpha() Mengembalikan true jika string memiliki minimal 1 karakter dan semua karakter adalah abjad dan false sebaliknya.
isdigit() Mengembalikan true jika string hanya berisi digit dan false sebaliknya.
isupper() Mengembalikan true jika string memiliki setidaknya satu karakter casing dan semua karakter casing ada dalam huruf besar dan false sebaliknya.
len(string) Mengembalikan panjang string.
replace(old, new [, max]) Menggantikan semua kemunculan lama dalam string dengan kejadian baru atau paling maksimal jika max diberikan.

PYTHON FILE HANDLING

Penanganan berkas merupakan bagian penting dari setiap aplikasi web. Python memiliki beberapa fungsi untuk membuat, membaca, memperbarui, dan menghapus berkas.
Fungsi utama untuk operasi file di Python adalah fungsi open(). Fungsi open() menggunakan dua parameter, nama berkas dan mode. Ada empat metode (mode) berbeda untuk membuka berkas antara lain:


"r" - Read - Nilai default. Membuka berkas untuk dibaca, kesalahan jika berkas tidak ada.
"a" - Append - Membuka berkas untuk ditambahkan, membuat berkas jika tidak ada.
"w" - Write - Membuka berkas untuk ditulis, membuat berkas jika tidak ada.
"x" - Create - Membuat berkas yang ditentukan, mengembalikan kesalahan jika berkas ada.

Kita dapat menentukan apakah file ditangani sebagai mode biner atau teks:


"t" - Text - Nilai default. Mode teks.
"b" - Binary - Mode binari (misal file gambar)

Open File

Fungsi open() mengembalikan objek file, dan memiliki metode read() untuk membaca isi file. Jika berkas berada di lokasi lain, kita harus menyebutkan letak spesifiknya. Jika file berada dalam folder yang sama dengan file coding, tinggal panggil nama file.


file1 = open("D:\\myfiles\welcome.txt")
print(file1.read())

file2 = open("demofile.txt")
print(file2.read())

Membuka file menggunakan "with":


with open("demofile.txt") as f:
    print(f.read())

Dengan menggunakan "with", kita tidak perlu khawatir untuk menutup file, pernyataan with otomatis menutup file.

Close File

Disarankan untuk selalu menutup berkas setelah selesai digunakan. Jika tidak menggunakan pernyataan with, kita harus menulis pernyataan close untuk menutup berkas:


f = open("demofile.txt")
print(f.readline())
f.close()

Write to an Existing File

Untuk menulis ke berkas yang sudah ada, Anda harus menambahkan parameter ke fungsi open():

  • "a" - Tambahkan - akan menambahkan ke akhir berkas
  • "w" - Tulis - akan menimpa konten yang sudah ada

Misal kita membuka file "demofile.txt" dan menambahkan isi ke file:


with open("demofile.txt", "a") as f:
  f.write("Now the file has more content!")

#open and read the file after the appending:
with open("demofile.txt") as f:
  print(f.read())

Delete a File

Untuk menghindari error, kita bisa memeriksa apakah berkas tersebut ada sebelum mencoba menghapusnya: 


import os
if os.path.exists("demofile.txt"):
    os.remove("demofile.txt")
else:
    print("The file does not exist")

Tugas 1: Mengecek Palindrom

Buatlah sebuah program untuk menerima inputan dua buah teks. Kemudian buat fungsi untuk mengecek apakah kedua teks tersebut merupakan Palindrom. Palindrom merupakan sebuah kata, bilangan, frasa, atau susunan karakter lain yang serupa jika dibaca dengan urutan terbalik ataupun tidak, seperti dalam kakak ataupun apa.
Contoh: "kasur ini rusak", jika susunan huruf dibalik maka "kasur ini rusak" (artinya ini adalah Palindrom).
