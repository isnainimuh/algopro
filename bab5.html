Dasar Teori

KONSEP PEMROGRAMAN MODULAR

Pemrograman modular adalah sebuah program yang terdiri dari beberapa modul atau subprogram. Ide utama dari pemrograman modular adalah membagi sebuah program yang kompleks ke dalam beberapa subprogram/modul yang lebih sederhana yang saling terkait satu dengan yang lain. Pemrograman modular memiliki beberapa manfaat diantaranya menyederhanakan pemrograman yang kompleks menjadi modul-modul sederhana, menghindari penulisan kode yang sama berulang-ulang, memudahkan pelacakan kesalahan, dan modifikasi dapat dilakukan tanpa mengganggu program keseluruhan. Modul atau subprogram dapat juga disebut sebagai fungsi atau prosedur.

Definisi Fungsi

Fungsi merupakan bagian dari kode program yang terpisah dari program utama. Bagain kode program apat dipanggil pada program utama dan ini di-execute jika diperlukan untuk melakukan tindakan khusus dalam program. Fungsi dibedakan menjadi user-define function (didefinisikan sendiri) dan built-in function (disediakan oleh compiler).
Deklarasi fungsi di python menggunakan kata kunci def. 


def namaFungsi (daftar parameter):
		blok kode program fungsi

Fungsi tanpa nilai balik

Terdapat dua macam fungsi yaitu fungsi dengan nilai kembalian (mengembalikan nilai ke pemanggil) atau fungsi tanpa nilai kembalian (return value). Berikut ini adalah contoh fungsi tanpa return value.


def cetak():
    print("Welcome to Python Jungle")

#memanggil fungsi di program utama
cetak()

Fungsi dengan nilai balik

Fungsi ini berguna untuk melakukan suatu proses yang dapat mengembalikan sebuah nilai ke pemanggilnya. Berikut ini adalah contohnya:


def cetak():
    return "Welcome to Python Jungle"

#memanggil fungsi di program utama
x=cetak()
print(x)

Perbedaan mendasar pada fungsi yang memiliki nilai balik adalah fungsi ini memakai sintaks return untuk mengirim value atau nilai ke pemanggilnya, sedangkan fungsi tanpa return value hanya mengeksekusi statement di dalamnya tanpa me-return nilai.

Parameter Fungsi

Parameter atau argument dalam fungsi digunakan untuk mengirim nilai dari program pemanggil ke dalam fungsi. Parameter ditulis Ketika parameter tersebut akan diolah di dalam fungsi. Contoh:


def penjumlahan(a, b):
    hasil = a + b
    return hasil

#memanggil fungsi di program utama
print(penjumlahan(9, 17))

Catatan:

  • Jika kita tidak mengetahui berapa banyak keyword argumen yang ada, tambahkan dua tanda bintang: ** sebelum nama parameter dalam definisi fungsi.
  • Dengan cara ini fungsi tersebut akan menerima kamus argumen, dan dapat mengakses item yang sesuai.
  • Di Python kita dapat mengeset nilai default untuk sebuah parameter fungsi, sehingga Ketika kita memanggil fungsi tanpe mendefinisikan parameternya, maka program akan mengambil parameter default.
  • Kita dapat mengirim tipe data apa pun dari argumen ke suatu fungsi (string, angka, list, tupple, dll.) dan itu akan diperlakukan sebagai tipe data yang sama di dalam fungsi.

Keyword *args dan **kwargs di Python

Keyword *args dan **kwargs di Python adalah sintaks khusus yang memungkinkan fungsi menerima sejumlah argumen variabel, membuat fungsi lebih fleksibel; *args mengemas argumen posisional tanpa kata kunci menjadi sebuah tuple, sedangkan **kwargs mengemas argumen kata kunci (keyword) menjadi sebuah dictionary (kamus), dengan * untuk args dan ** untuk kwargs, dan keduanya bisa diganti namanya.

*args (Argumen Posisi Variabel)

  • Berfungsi menerima sejumlah argumen posisi (non-keyword) yang tidak diketahui jumlahnya.
  • Luarannya berupa argumen yang dikemas menjadi tuple di dalam fungsi.

**kwargs (Argumen Kata Kunci Variabel)

  • Berfungsi menerima sejumlah argumen kata kunci (keyword arguments) yang tidak diketahui jumlahnya.
  • Argumen tersebut dikemas menjadi dictionary di dalam fungsi, di mana key adalah nama argumen dan value adalah nilainya.

Urutan dan Penggunaan

  • Urutan dalam definisi fungsi adalah: argumen wajib, *args, lalu **kwargs.
  • Kita bisa menggunakan keduanya bersamaan dalam satu fungsi untuk fleksibilitas maksimum, misalnya: def fungsi_lengkap(arg_wajib, *args, **kwargs).
  • Nama args dan kwargs bersifat konvensional, kita bisa menggantinya dengan nama lain selama tanda bintang (* atau **) tetap ada, contoh: def fungsi(*param_posisi, **param_keyword).
